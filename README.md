# PromptImageScrape

CLI application that crawls any domain, scrapes all images, embeds them with **jina-clip-v2**, and lets you search by text prompt.

## Status

- [x] Crawl package (spiders, extractors, pipelines, PDF support, sitemap parsing, JS detection)
- [ ] Storage adapters
- [ ] Indexing pipeline
- [ ] Unified CLI
- [ ] Documentation
